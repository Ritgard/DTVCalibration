# Калибровка и проверка модулей ДТВ
Версия **1.0.0**

Базовый функционал:
- Опрос модулей ДТВ
- Опрос эталона температуры HMP155 или Rotronik
- Занос кодов от модулей ДТВ в таблицу, в соответствии с температурой
- Загрузка данных из excel файла
- Занос температуры от модулей при проверке, в таблицу, в соответствии с температурой
- Скриншот рабочей области
- Рассчет коэффициентов
- Сохранение файлов калибровки
- Рассчет погрешности
